# Get color palette for the smooth values

Get color palette for the smooth values

## Usage

``` r
.get_smooth_palette(no_breaks)
```

## Arguments

- no_breaks:

  numeric number of breaks on scale

## Value

gDR palette for smooth values with given `no_breaks`

## Examples

``` r
if (FALSE) { # \dontrun{
.get_smooth_palette(25)
} # }
```
