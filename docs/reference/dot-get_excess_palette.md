# Get color palette for the excess values

Get color palette for the excess values

## Usage

``` r
.get_excess_palette(no_breaks)
```

## Arguments

- no_breaks:

  numeric number of breaks on scale

## Value

gDR palette for excess values with given `no_breaks`

## Examples

``` r
if (FALSE) { # \dontrun{
.get_excess_palette(20)
} # }
```
