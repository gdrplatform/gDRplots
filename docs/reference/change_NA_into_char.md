# Change NA into given string

Change NA into given string

## Usage

``` r
change_NA_into_char(x, lbl_NA = "NA")
```

## Arguments

- x:

  vector with items suspected of being NA

- lbl_NA:

  string - replacement for NA - as default "NA"

## Value

character (for NA -\> given string)
